# Contributing

This is a fork. Upstream contributions belong at
<https://github.com/yomidevs/yomitan>.

## Local dev loop

```bash
npm ci                  # once
npm run test:fast       # eslint + tsc + unit + JSON format
npm run build           # before testing in a browser
```

The `ext/` directory is a live, loadable unpacked Chrome extension — for most
edits you do not need to rebuild; just reload the extension at
`chrome://extensions`.

## Conventions

- License header (GPL-3.0-or-later) is enforced by `eslint-plugin-header` on
  every JS file. Keep it intact.
- Modern ES modules only (`const`/`let`, `async`/`await`, arrow functions).
- Do not hand-edit `ext/manifest.json` — it is generated by `npm run build`
  from `dev/data/manifest-variants.json`.
- `husky` + `lint-staged` format markdown on commit.

## Tests

- `npm run test:fast` — fast loop (eslint + tsc + unit + JSON).
- `npm test` — full pipeline (also CSS, HTML, markdown, build dry-run).
- `npm run test:unit -- path/to/file.test.js` — single file.
- `npm run test:unit -- -t "pattern"` — by name.

## Releasing

Push a tag matching `v*` to trigger the release workflow. See `README.md`.
